# Установка и запуск приложения, а также выполнение тестов

### 1. Клонирование репозитория

Скопируйте удалённый репозиторий с помощью команды:  
`git clone https://github.com/NikolayGainulin/-diploma/`

### 2. Открытие проекта

Загрузите склонированный проект в среду разработки Android Studio.

### 3. Подготовка устройства к тестированию

- Запустите виртуальный эмулятор либо подключите реальный смартфон/планшет.
- В настройках устройства выберите русский язык интерфейса.
- Убедитесь, что для входа в приложение используются следующие данные:
    - **Логин**: `login2`
    - **Пароль**: `password2`

### 4. Переход в папку проекта

Откройте директорию `fmh_android_15_03_24` внутри проекта.

### 5. Запуск тестов

- Выполните в терминале следующую команду:  
  `./gradlew connectedAndroidTest`

- Если появляется сообщение `zsh: permission denied: ./gradlew`, необходимо сделать файл исполняемым:  
  `chmod +x ./gradlew`

- Дождитесь, пока все тесты завершат своё выполнение.

### 6. Генерация отчёта через AllureReport

#### Подготовка Allure

В файле `allure.properties` (папка `androidTest/resources`) добавьте строку:  
`allure.results.useTestStorage=true`

#### Добавление библиотеки

В файл `build.gradle` вашего приложения пропишите зависимость:  
`androidTestUtil "androidx.test:orchestrator:1.4.2"`

#### Выгрузка полученных данных

С помощью инструмента `Device File Explorer` в Android Studio скачайте папку с результатами тестирования с устройства:  
`/sdcard/googletest/test_outputfiles/allure-results`

#### Построение отчёта

Переместите выгруженную папку на ваш компьютер. Откройте терминал в папке, которая находится **на уровень выше** каталога с результатами, и выполните:  
`allure serve`

- Дождитесь автоматического построения отчёта — он откроется в браузере.
